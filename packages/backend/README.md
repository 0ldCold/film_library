# BackEnd

## Стек

- Express.js
- TypeScript
- PostgreSQL
- TypeORM
- TSOA
- class-validator

## Запуск

### В dev-mode

1. `yarn install`
2. `docker-compose up`
3. `yarn run seed:fresh`
4. `yarn run dev`

### В prod-mode

1. `yarn install`
2. `yarn run start:prod`

### CI/CD

1. `yarn install --immutable`
2. `yarn run types`
3. `yarn run test`

### Проверка кода:

1. `yarn run check`
