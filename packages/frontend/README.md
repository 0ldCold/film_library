# FrontEnd

## Стек

- React
- TypeScript
- Next.js
- ReactQuery
- i18n

Проект строится по архитектуре [Feature-Sliced Design](https://feature-sliced.design/ru/docs/get-started/overview)

## Запуск

### В dev-mode

1. `yarn install`
2. `yarn run dev`

### В prod-mode

1. `yarn install`
2. `yarn run prodmode`

### CI/CD

1. `yarn install --immutable`
2. `yarn run types`
3. `yarn run lint:check`
4. `yarn run prettier:check`
5. `yarn run build`

### Проверка кода:

1. `yarn run check`
